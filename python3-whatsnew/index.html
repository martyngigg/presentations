Python 3

What's New!

Print Function

Additional arguments, e.g. sep & end 


>>> print('hello', 'world', sep='.')
hello.world
>>> print('hello', end=' '); print('world')
hello world

String Interpolation: f-strings

Pre python 3.6


>>> value = 4 * 20
>>> 'The value is {value}.'.format(value=value)
 'The value is 80.'

Python >= 3.6


>>> value = 4 * 20
>>> f'The value is {value}.'
 'The value is 80.'

Braces can contain any expressions


>>> f'The value is {4 * 20}.'
 'The value is 80.'

Advanced Unpacking

Python 2


>>> first, second, _, _, _ = range(5)

Python 3


>>> first, second , *_ = range(5)  # python 3
>>> first, *_, last = range(5)  # python 3

Division now float by default


>>> 1 / 2
0.5
>>> 1 // 2  # // access integer division
0

Keywords allowed after *varargs


def sortwords(*wordlist, case_sensitive=False):

>>> def sub(x, y, *, reverse):

Libraries

Recursive option in glob


>>> import os
>>> import glob
>>> glob.glob(os.path.join('data', '**', '*.fits'), recursive=True)
['data/image.fits', 'data/subset1/a.fits', 'data/subset1/b.fits',
 'data/subset1/c.fits', 'data/subset2/d.fits', 'data/subset2/e.fits']

Numpy 1.10: @ performs matrix multiplication


>>> import numpy as np
>>> x = np.array([[1, 2], [3, 4]])
>>> y = np.array([[3, 2], [2, -1]])
>>> x @ y
array([[ 7,  0],
       [17,  2]])
